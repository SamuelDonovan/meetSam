Python
======

Installation From Pypi
----------------------

.. code-block:: console

   (.venv) $ pip install meetSam 

Console Usage
-------------

.. code-block:: console

   (.venv) $ python -m meetSam 

Python Usage
-------------

.. code-block:: console

   python
   >>> import meetSam
   >>> meetSam.main()
   >>> meetSam.site()
   >>> help(meetSam.meet_sam)
